Windows重装系统及如何激活

发表于2026-05-30更新于2026-06-05
经验分享系统

Windows重装系统及如何激活

余逢2026-05-302026-06-05

使用 WinPE 制作系统盘并安装 Windows 系统完全教程

本教程面向零基础用户，从准备工作到系统安装完成，每一步都有详细说明。
阅读时间约 15 分钟，实际操作约 30-60 分钟。

1. 认识 PE 和装机流程

什么是 PE？

PE（Preinstallation Environment，预安装环境）是一个微型操作系统，可以装在 U 盘里。
它的作用是：在你的电脑硬盘上没有系统（或系统坏了）的情况下，让你能启动电脑，
然后用它来给硬盘分区、安装正式的 Windows 系统。

你可以把它理解为：PE 是一个”临时工”系统，帮你把正式的 Windows 安装到硬盘上。

整体流程概览

准备 U 盘 → 制作 PE 启动盘 → 把系统镜像放进 U 盘 → 设置电脑从 U 盘启动 → 进入 PE → 给硬盘分区 → 运行安装程序 → 完成安装 → 设置系统

2. 准备工作（你需要什么）

硬件准备

物品 要求 说明
U 盘 容量 ≥ 8GB，推荐 16GB 或更大 会被格式化，里面的文件会全部丢失！
可上网的电脑 任意一台能正常上网的电脑 用来下载工具和制作启动盘（可以是别人的电脑）
要装系统的电脑 就是你要重装/新装系统的那台 确保能插 U 盘，键盘能用

软件准备（后面会教你下载）

  • 微PE 工具箱（WePE）—— 用来制作 PE 启动 U 盘
  • Windows 系统镜像（ISO 文件）—— 你要安装的操作系统

重要提醒

⚠️ 数据备份！数据备份！数据备份！

安装系统通常需要格式化硬盘，这会永久删除硬盘上的所有数据。
如果你的电脑里有重要文件（照片、文档、视频等），请先用另一个 U 盘或移动硬盘备份出来！

格式化后数据无法恢复（普通手段下），请务必先备份！

3. 下载所需工具和镜像

3.1 下载微 PE 工具箱（WePE）

微 PE 是目前最流行、最易用的 PE 制作工具，完全免费。

  1. 打开浏览器，访问官网：https://www.wepe.com.cn
  2. 在首页找到下载按钮，点击下载最新版本
  3. 当前推荐下载 64 位版本（V2.3 或更新版本）
  4. 文件大小约 200-300MB，下载后备用

💡 提示： 如果官网打不开，可以在百度搜索”微PE工具箱下载”，但请认准官网或可信下载站，
避免下载到捆绑了流氓软件的版本。

3.2 下载 Windows 系统镜像（ISO 文件）

推荐来源（最安全）：

  • 微软官网：https://www.microsoft.com/zh-cn/software-download/

    • Windows 11：https://www.microsoft.com/zh-cn/software-download/windows11

    • Windows 10：https://www.microsoft.com/zh-cn/software-download/windows10

  • 在页面中找到”下载工具”或”下载 ISO”选项

  • 选择你需要的版本（推荐 Windows 11 专业版 或 家庭版）

备选来源：

  • “我告诉你”网站（https://msdn.itellyou.cn）—— 提供微软原版镜像下载
  • HelloWindows（https://hellowindows.cn）—— 同样提供原版镜像

💡 关于版本选择的建议：

  • 2020 年后的电脑：推荐 Windows 11
  • 较老的电脑（2015 年前）：推荐 Windows 10
  • 不确定选哪个：先试 Windows 11，装不了再换 Windows 10

⚠️ 切记不要从不明小站下载”Ghost版””精简版””装机版”系统！
这些系统可能被植入广告、后门、病毒，非常不安全。请下载微软原版镜像。

4. 制作 PE 启动 U 盘

这一步是把你的普通 U 盘变成一个能启动电脑的”PE 系统盘”。

4.1 插入 U 盘

  1. 把 U 盘插入电脑的 USB 接口
  2. 确认 U 盘里没有重要数据（制作过程会格式化 U 盘，数据全部清除）
  3. 等待电脑识别到 U 盘（通常会有提示音或在”此电脑”中出现盘符）

4.2 打开微 PE 工具箱

  1. 找到刚才下载的 WePE 文件（通常在”下载”文件夹中）
  2. 右键 → 以管理员身份运行
  3. 如果弹出 UAC（用户账户控制）提示，点”是”

4.3 选择安装到 U 盘

  1. 打开后，你会看到微 PE 的主界面
  2. 点击右下角的 U 盘图标（通常写着”安装PE到U盘”或类似字样）
  3. 进入安装设置界面

4.4 设置安装参数

在安装界面中，你需要确认以下设置：

设置项 推荐选择 说明
选择设备 选择你的 U 盘 ⚠️ 一定要选对！看清楚盘符和容量，别选成移动硬盘
格式化方式 exFAT 如果 U 盘是 32GB 及以上选 exFAT；小于 32GB 也可以选 FAT32
UEFI 引导 勾选 现在绝大多数电脑都是 UEFI 模式
Legacy 引导 可以一起勾选 兼容老电脑，不影响新电脑

⚠️ 再次确认：选对 U 盘！ 错误的选择可能导致其他磁盘数据丢失。

4.5 开始制作

  1. 确认设置无误后，点击 “立即安装到U盘” 按钮
  2. 弹出警告提示”将格式化U盘”，确认后点”是”
  3. 等待进度条走完，通常需要 1-3 分钟
  4. 看到 “安装完成” 提示后，点确定

4.6 验证制作成功

制作完成后：

  1. 打开”此电脑”（或”我的电脑”）
  2. 你会看到 U 盘变成了两个分区：
    • 一个叫 “EFI” 或 “WEPE” 的小分区（PE 系统文件）
    • 一个正常的 U 盘分区（剩余空间，用来放系统镜像）

✅ 到这一步，你的 PE 启动 U 盘就制作完成了！

5. 下载并放入系统镜像

5.1 找到系统镜像文件

如果你在第 3 步已经下载好了 Windows 的 ISO 文件，现在找到它。
ISO 文件通常文件名类似：

  • Win11_24H2_Chinese_Simplified_x64.iso
  • zh-cn_windows_10_business_editions_xxx.iso

文件大小通常在 4-6 GB 左右。

5.2 把镜像复制到 U 盘

  1. 打开”此电脑”
  2. 找到 U 盘的数据分区（不是 EFI/WEPE 那个小分区，是大的那个）
  3. 直接把 ISO 文件复制粘贴到 U 盘的根目录（最外层，不要放进任何文件夹里也可以）

💡 提示： 如果你打算给 GPT 分区的硬盘装系统，且 U 盘剩余空间足够，
也可以直接解压 ISO 文件到 U 盘，这样在 PE 里可以直接运行 setup.exe。

5.3 安全弹出 U 盘

  1. 文件复制完成后，右键点击任务栏右下角的 U 盘图标
  2. 选择 “安全弹出” 或 “弹出 [U盘名]”
  3. 等待提示”可以安全移除”后，拔出 U 盘

⚠️ 不要直接拔 U 盘！可能导致文件损坏。

6. 设置电脑从 U 盘启动

现在要把 U 盘插到要装系统的那台电脑上，然后设置它从 U 盘启动。

6.1 插入 U 盘

  1. 把制作好的 PE 启动 U 盘插入要装系统的电脑
  2. 建议插在机箱后面的 USB 接口（直接在主板上，兼容性更好）
  3. 如果是笔记本，随便哪个 USB 口都行

6.2 进入 BIOS / 启动菜单

有两种方式可以设置从 U 盘启动：

方式一：使用启动快捷键（推荐，最简单）

  1. 先插好 U 盘
  2. 按下电脑的开机键
  3. 在出现品牌 Logo 的一瞬间，连续、快速地按启动菜单快捷键

常见品牌的启动菜单快捷键：

品牌 启动菜单快捷键 备注
联想（Lenovo） F12 部分型号是 F12 或 Fn+F12
戴尔（Dell） F12 开机后立即连续按
华硕（ASUS） F8 或 Esc 部分主板是 F8
惠普（HP） F9 或 Esc 先按 Esc 再选 F9
宏碁（Acer） F12 需要在 BIOS 中先开启 F12 Boot
微星（MSI） F11
华为（Huawei） F12 笔记本较多
小米（Xiaomi） F12
苹果 Mac 装 Windows Option (Alt) 开机时长按
  1. 按对了会出现一个 启动设备选择菜单
  2. 用键盘上下箭头选择你的 U 盘（通常显示 U 盘品牌名或”USB”字样）
  3. 按 Enter 确认

方式二：进入 BIOS 设置（适用于方式一不管用时）

  1. 开机时连续按 BIOS 进入键
品牌 BIOS 进入键
联想 F2 或 Fn+F2
戴尔 F2
华硕 F2 或 Del
惠普 F10 或 Esc
宏碁 F2 或 Del
微星 Del
  1. 进入 BIOS 后，找到 Boot（启动） 选项卡
  2. 在启动顺序（Boot Priority / Boot Order）中，把 U 盘 移到第一位
  3. 如果有 Secure Boot（安全启动） 选项，建议先关闭它
  4. 如果有 Boot Mode 选项，确认是 UEFI（新电脑）或 Legacy（老电脑）
  5. 按 F10 保存并退出（Save & Exit）

💡 小贴士：

  • 按快捷键的时机很重要：要在品牌 Logo 出现时快速连续按，不要太早也不要太晚
  • 如果错过了，长按电源键强制关机，再重新开机重试
  • 如果是新电脑装 Windows 11，通常选 UEFI 模式 + GPT 分区

6.3 UEFI 与 Legacy 的区别（了解即可）

模式 分区表 适用场景
UEFI GPT 新电脑（2015年后），支持 2TB 以上硬盘，Windows 11 必须
Legacy MBR 老电脑，最大支持 2TB 硬盘

💡 如果不确定，新电脑选 UEFI，老电脑选 Legacy。

7. 进入 PE 系统

7.1 成功从 U 盘启动后

  1. 电脑会从 U 盘加载 PE 系统
  2. 你会看到一个启动菜单（类似 Windows 的界面）
  3. 通常选择 “Windows PE (x64)” 或 “进入PE系统”
  4. 等待加载，通常需要 30秒到2分钟

7.2 PE 系统桌面

加载完成后，你会看到一个类似 Windows 的桌面。恭喜你，已经成功进入 PE 了！

桌面上通常会有这些常用工具：

  • 分区工具（DiskGenius 或 分区助手）
  • Windows 安装器（WinNTSetup 或 CGI）
  • 引导修复工具
  • 系统备份/还原工具
  • 其他维护工具

💡 注意： PE 里的操作不会影响到你 U 盘里的文件，但会对电脑硬盘进行操作，
所以每一步都要看清楚再点。

8. 给硬盘分区（非常重要）

⚠️ 这一步会清除硬盘上的所有数据！请确认已备份重要文件！

分区就是把硬盘划分成不同的区域，比如 C 盘（系统盘）、D 盘（数据盘）等。

8.1 打开分区工具

  1. 在 PE 桌面上找到 DiskGenius（或”分区工具”）
  2. 双击打开

8.2 认识分区工具界面

打开后你会看到：

  • 上方：硬盘列表（硬盘0、硬盘1……）
  • 中间：分区列表（显示每个硬盘上的分区情况）
  • 下方：操作按钮和菜单

8.3 选择分区方案

方案 A：全新分区（推荐，适用于新硬盘或想全部重来的情况）

  1. 在硬盘列表中选中你要安装系统的硬盘（通常是”硬盘0”，看容量判断）

  2. 点击菜单栏的 “硬盘” → “转换分区表类型为GUID格式”（即 GPT）

    • 如果你的电脑是 UEFI 模式，选 GPT
    • 如果是 Legacy 模式，选 MBR

  3. 点击 “快速分区” 按钮（或菜单中的”硬盘”→”快速分区”）

  4. 在弹出的分区设置窗口中：

推荐分区方案（UEFI + GPT）：

分区序号 卷标 大小 文件系统 说明
ESP ESP 300-500 MB FAT32 UEFI 引导分区，自动创建
MSR MSR 16 MB — 微软保留分区，自动创建
1 系统 100-200 GB NTFS C 盘，装系统和常用软件
2 数据 剩余全部 NTFS D 盘，放个人文件和资料

💡 关于 C 盘大小的建议：

  • 最低 60GB（勉强够用，但会很快变满）
  • 推荐 100-200GB（装完系统+常用软件后还很充裕）
  • 如果硬盘只有 128GB SSD：就全部给 C 盘，不分 D 盘
  1. 确认设置后，点击 “确定”
  2. 弹出警告”将清除所选硬盘所有分区”，确认后点 “是”
  3. 等待分区完成

如果是 Legacy + MBR 方案：

  • 主分区（C 盘）：100-200GB，NTFS，设为”活动分区”
  • 逻辑分区（D 盘等）：剩余空间，NTFS

方案 B：只格式化 C 盘（适用于重装系统但想保留其他分区数据）

  1. 在分区列表中找到 C 盘（通常是第一个主分区）
  2. 右键点击 C 盘 → “格式化当前分区”
  3. 文件系统选择 NTFS
  4. 点击 “格式化” → 确认

⚠️ 格式化 C 盘前请确认 D 盘等其他分区的数据是你想保留的！

8.4 保存分区更改

  1. 分区操作完成后，点击工具栏的 “保存更改” 按钮
  2. 确认所有操作
  3. 等待执行完成

✅ 分区完成后，你现在有了一个（或多个）空白的、格式化好的分区，可以安装系统了。

9. 安装 Windows 系统

方法一：使用 WinNTSetup 安装（推荐）

这是 PE 下最常用的安装方式，操作简单且可靠。

9.1 打开 Windows 安装器

  1. 回到 PE 桌面
  2. 找到并打开 WinNTSetup（或叫”Windows 安装器”）

9.2 挂载系统镜像

如果你的 U 盘里放的是 ISO 文件，需要先”挂载”它（就像打开一个压缩包）：

  1. 打开”此电脑”（在 PE 里也有）
  2. 找到 U 盘盘符（通常显示为 D: 或 E: 等，注意看容量判断）
  3. 找到 ISO 文件，双击打开
  4. 系统会自动挂载，会出现一个新的虚拟光驱盘符（比如 G:）
  5. 打开这个虚拟光驱盘符，里面应该能看到 sources 文件夹和 setup.exe 等文件

💡 如果双击 ISO 没反应： 可以用 PE 里自带的虚拟光驱工具（如 ImDisk）手动挂载。
或者如果之前是解压到 U 盘的，直接找到解压后的文件夹即可。

9.3 配置安装参数

在 WinNTSetup 界面中，你需要填写以下内容：

第一行：选择 Windows 安装文件的位置

  • 点击 “选择” 按钮
  • 浏览到挂载的虚拟光驱盘符（或 U 盘中的解压文件夹）
  • 进入 sources 文件夹
  • 选择 install.wim 或 install.esd 文件
  • 点击打开

第二行：选择引导驱动器的位置

  • 通常会自动识别
  • UEFI 模式：选择那个 300MB 左右的 ESP 分区（FAT32 格式）
  • Legacy 模式：选择硬盘的主分区（C 盘）或者系统保留分区
  • 如果不确定，通常保持默认即可

第三行：选择安装驱动器的位置（系统安装到哪里）

  • 点击 “选择” 按钮
  • 选择你刚才分好的 C 盘（系统分区）
  • ⚠️ 千万选对！不要选成 U 盘或数据分区！

版本选择：

  • 如果镜像包含多个版本（家庭版、专业版等），在下拉菜单中选择你需要的版本
  • 推荐选择 Windows 11 专业版 或 Windows 10 专业版

9.4 开始安装

  1. 确认所有选项无误后，点击 “开始安装” 按钮
  2. 弹出确认对话框，通常有以下选项：
    • “安装后自动重启” → 可以勾选
    • “添加引导” → 确保勾选
  3. 点击 “确定”
  4. 等待文件释放和安装，这个过程通常需要 5-15 分钟
    • 进度条会显示安装进度
    • 此时不要拔 U 盘、不要断电

9.5 安装完成后的第一次重启

  1. 文件释放完成后，电脑会自动重启
  2. ⚠️ 重要：在重启时拔掉 U 盘！
    • 如果不拔，电脑可能会再次进入 PE 而不是硬盘上的新系统
    • 如果忘记拔了也没关系，进入 PE 后拔掉再重启即可
  3. 重启后，电脑会从硬盘启动，开始 Windows 的初始配置

方法二：使用系统自带的 Setup.exe 安装（备选方案）

如果你更习惯传统的安装方式：

  1. 在 PE 中打开”此电脑”
  2. 找到挂载的系统镜像盘符
  3. 直接双击运行 setup.exe
  4. 按照向导操作：
    • 语言和区域设置 → 点”下一步”
    • 点击 “现在安装”
    • 输入产品密钥（可以点”我没有产品密钥”跳过，之后再激活）
    • 选择要安装的版本
    • 接受许可条款
    • 选择 “自定义：仅安装 Windows（高级）”
    • 选择 C 盘分区，点”下一步”
  5. 等待安装完成

💡 这种方式有时候在 PE 下可能不支持，如果遇到问题就用方法一（WinNTSetup）。

10. 系统初始设置

重启后，Windows 会开始初始设置流程（称为 OOBE - Out of Box Experience）。

10.1 区域设置

  1. 选择国家/地区：中国
  2. 点击 “是”

10.2 键盘布局

  1. 选择键盘布局：微软拼音
  2. 点击 “是”
  3. 是否添加第二种键盘布局：点 “跳过”

10.3 网络连接

  1. 连接到 Wi-Fi 或插网线
  2. 如果暂时不想联网：
    • Windows 11：按 Shift + F10，输入 OOBE\BYPASSNRO，回车，电脑会重启，重启后有”我没有 Internet 连接”选项
    • Windows 10：可以直接点”我没有 Internet 连接”

10.4 账户设置

如果联网了：

  • 可以登录微软账号（推荐，能同步设置和 OneDrive）
  • 也可以选择创建本地账户

如果没联网或想用本地账户：

  • Windows 10：直接选择”脱机账户”→”有限体验”
  • Windows 11：可能需要通过上面的绕过命令才能使用本地账户

10.5 设置用户名和密码

  1. 输入你想要的用户名（比如你的名字或昵称）
  2. 设置密码（可以留空不设，但建议设置）
  3. 设置安全问题（如果设了密码的话）

10.6 隐私设置

  1. 根据自己的偏好选择各项隐私设置
  2. 建议全部关闭，以后需要时再开
  3. 点击 “接受”

10.7 等待系统配置完成

  1. Windows 会进行最后的配置
  2. 可能需要几分钟时间
  3. 最终你会看到 Windows 桌面 🎉

✅ 恭喜！Windows 系统安装成功！

11. 安装驱动和常用软件

11.1 安装驱动程序

驱动程序是让硬件（显卡、声卡、网卡等）正常工作的软件。

联网后 Windows 会自动安装大部分驱动，但有些驱动需要手动安装。

方法一：Windows Update 自动安装

  1. 打开 设置 → Windows 更新（Win10：更新和安全）
  2. 点击 “检查更新”
  3. Windows 会自动下载并安装驱动
  4. 安装完后可能需要重启，反复检查更新直到没有新更新

方法二：使用驱动管理工具（如果某些驱动没装上）

  • 驱动精灵（http://www.drivergenius.com）
  • 驱动人生（https://www.160.com）
  • 联想驱动管理（联想电脑专用）
  • 品牌官网驱动下载页面

💡 提示： 如果你电脑有独立显卡，建议去显卡官网下载最新驱动：

  • NVIDIA 显卡：https://www.nvidia.cn/drivers/
  • AMD 显卡：https://www.amd.com/zh-hans/support
  • Intel 核显：通过 Windows Update 或 Intel 官网

确认驱动是否安装完整

打开 设备管理器（右键”此电脑”→”管理”→”设备管理器”）：

  • 如果某个设备前面有 黄色感叹号 ⚠️，说明该设备的驱动没装好
  • 右键该设备 → “更新驱动程序” → “自动搜索驱动”

12. 常见问题与故障排除

Q1：按了启动快捷键没反应 / 进不了启动菜单

可能原因和解决方法：

  • 按键时机不对：在品牌 Logo 出现时快速连续按，不要只按一下
  • 部分笔记本需要先关闭 快速启动：在 BIOS 中找 “Fast Boot” 选项关闭
  • 尝试用 BIOS 设置方式（方式二）修改启动顺序
  • U 盘可能没有正确制作：回到别的电脑上重新制作

Q2：进入 PE 后看不到硬盘

可能原因和解决方法：

  • 硬盘模式问题：进入 BIOS，找到 SATA Mode，切换为 AHCI 模式
  • NVMe SSD 驱动问题：使用最新版的微 PE（V2.3+），老版本可能不支持 NVMe
  • 硬盘连接问题：台式机检查 SATA 线是否插好；笔记本可能需要专业检查

Q3：安装过程中报错”Windows 无法安装到这个磁盘”

可能原因和解决方法：

  • 分区表类型不对：UEFI 模式需要 GPT 分区表，Legacy 需要 MBR
    • 在 PE 中用 DiskGenius 转换分区表类型
  • 硬盘有旧的分区信息残留：重新快速分区

Q4：安装完成后重启蓝屏

可能原因和解决方法：

  • 驱动不兼容：进入安全模式（开机时反复按 F8），卸载最近安装的驱动
  • 硬盘模式不对：BIOS 中的 SATA Mode 需要和安装时一致
  • 引导损坏：用 PE 启动，使用”引导修复”工具修复

Q5：重启后没有进入 Windows 安装，又进了 PE

原因： U 盘没拔！
解决： 拔掉 U 盘，再重新开机。

Q6：装完系统后没有网络

可能原因和解决方法：

  • 网卡驱动没装：用别的电脑下载对应品牌和型号的网卡驱动，用 U 盘拷过来安装
  • 或者用手机 USB 网络共享：手机连 Wi-Fi，用 USB 线连接电脑，在手机设置中开启”USB 网络共享”

Q7：系统装好后没有激活

解决方法：

  • 如果有正版密钥：打开 设置 → 系统 → 激活，输入产品密钥
  • 如果是品牌电脑：通常内置了 OEM 密钥，联网后会自动激活
  • 购买正版密钥：微软官网或授权经销商
